## Getting Started

To get Bandwagon up and running follow these simple steps.



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/RMcCurdy/Bandwagon.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
